# Rutina Diaria - Enfoque y Disciplina

Mini app web personal para gestionar mi rutina diaria, hábitos, notas, cronómetro, sonidos y respaldo de progreso.

## Descripción

Esta app fue creada como una herramienta personal de productividad para organizar el día de forma visual, sencilla y minimalista.

Permite llevar seguimiento diario de tareas, escribir reflexiones personales, medir tiempo de enfoque con cronómetro y conservar una racha de días completados.

## Funciones principales

- Navegación por días.
- Checklist de rutina diaria.
- Barra de progreso del día.
- Racha de días completados.
- Cronómetro de enfoque.
- Bloc de notas por fecha.
- Sonidos suaves de interacción.
- Botón para activar o desactivar sonidos.
- Exportar respaldo en archivo JSON.
- Importar respaldo para recuperar datos.
- Diseño responsive tipo app móvil.

## Versiones

### v1.0 - Versión estable inicial

Incluye:
- Navegación por días.
- Racha.
- Cronómetro.
- Bloc de notas.
- Guardado local en el navegador.

### v1.1 - Versión con sonidos

Incluye:

### v1.2 - Versión con respaldo

Incluye:
- Exportar respaldo en JSON.
- Importar respaldo desde JSON.
- Recuperación de días, notas, tareas, cronómetros y preferencias.
- Mantiene sonidos y navegación por días.

## Uso

La app funciona directamente desde GitHub Pages.

Los datos se guardan en el almacenamiento local del navegador, por lo que se recomienda usar siempre el mismo dispositivo y navegador.

Para mayor seguridad, se puede usar la función de exportar respaldo.
